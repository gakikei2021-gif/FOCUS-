# Study Timer
Aka "Desk Guard", is a study timer for focus. Upon visiting the website, you will be asked to input the time you would like to focus. There are buttons for how long you want to focus: 25m, 45m, 60m, or 90m, the time is also customisable.
The unique thing here is that it requires you a password: you have to input the password before starting, and you will be asked to input again to exit and end the timer early. The exit button will ONLY show up after 5 minutes of focus, while also enabling the escape key, which will also be blocked for the first 5 minutes of focus.

Have fun studying:)
